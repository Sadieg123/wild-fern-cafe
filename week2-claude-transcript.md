# Claude Code Transcript – Week 2 Exercise
## DIG 4503 – Static Website with Claude Code

### Project Overview
Fictional business: Wild Fern Café  
Business type: Cozy neighborhood café  

This project was built using Claude Code to generate and iteratively refine a multi-page static website, including HTML structure, CSS styling, navigation, and responsive design.

---

## Initial Prompt – Site Setup

Create a multi-page static website for a fictional small business called 
"Wild Fern Café", a cozy neighborhood café.

I need the following files:
- index.html (home page)
- about.html
- menu.html
- contact.html
- styles.css

Include:
- A navigation menu on all pages linking to every page
- Clean, professional HTML structure
- A responsive layout suitable for mobile and desktop
- Link all pages to the same styles.css file
- Use semantic HTML and realistic content

---

## Follow-Up Prompt – Home Page

Update index.html to include:
- A hero section with the business name, tagline, and call-to-action
- An overview section (2–3 sentences)
- A “Why Choose Us” section with three reasons
- A preview section linking to the Menu and About pages

---

## Follow-Up Prompt – About Page

Update about.html with:
- A page header introducing the café
- A section telling the story of Wild Fern Café (3–4 sentences)
- A mission or values section
- A short section about atmosphere and community focus

---

## Follow-Up Prompt – Menu Page

Update menu.html to include:
- A clear page heading
- A brief menu introduction
- Three menu categories (Coffee, Breakfast, Lunch)
- 5–10 menu items with short descriptions

---

## Follow-Up Prompt – Contact Page

Update contact.html with:
- Contact details (fictional address, phone, email)
- Business hours
- A centered contact form with name, email, and message fields
- No backend functionality required

---

## Styling & UX Improvements

Prompts included:
- Make the navigation sticky at the top of the page
- Add more spacing between sections
- Center the main content
- Improve mobile responsiveness

---

## Debugging & Navigation Fix

Prompt used to resolve an error:
I’m getting an ERR_FILE_NOT_FOUND error when clicking the Home link from other pages.
Please review and fix the navigation so all links use correct relative paths.

---

## Final Review Prompt

Review the entire site and ensure:
- Navigation works on all pages
- Each page has at least 3 sections
- Tone is professional and realistic
- Site is responsive on mobile and desktop

---

## Outcome

Claude Code was used to:
- Generate all HTML and CSS files
- Write realistic business content
- Create responsive layouts
- Debug navigation issues
- Improve spacing, layout, and usability
